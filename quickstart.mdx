This guide walks you through cloning and running the portfolio locally, then building it for production.

## Prerequisites

- [Node.js](https://nodejs.org) v18 or later
- [Git](https://git-scm.com)

## Run locally

<Steps>
  <Step title="Clone the repository">
    ```bash
    git clone https://github.com/Iceheop/rsantana.git
    cd rsantana
    ```
  </Step>

  <Step title="Install dependencies">
    <CodeGroup>
      ```bash npm
      npm install
      ```
      ```bash pnpm
      pnpm install
      ```
      ```bash yarn
      yarn install
      ```
    </CodeGroup>
  </Step>

  <Step title="Start the development server">
    <CodeGroup>
      ```bash npm
      npm run dev
      ```
      ```bash pnpm
      pnpm dev
      ```
      ```bash yarn
      yarn dev
      ```
    </CodeGroup>

    <Note>
      Astro starts on **http://localhost:4321** by default. Open this URL in your browser to see the portfolio.
    </Note>
  </Step>

  <Step title="Explore the pages">
    The site has two pages:

    | URL | Page |
    |---|---|
    | `http://localhost:4321/` | Home — shows the developer bio and project grid |
    | `http://localhost:4321/projects` | All projects with breadcrumb navigation |
  </Step>
</Steps>

## Build for production

<Steps>
  <Step title="Run the build command">
    <CodeGroup>
      ```bash npm
      npm run build
      ```
      ```bash pnpm
      pnpm build
      ```
      ```bash yarn
      yarn build
      ```
    </CodeGroup>

    Astro outputs the static site to the `dist/` directory. Content collection frontmatter is validated against the Zod schema at build time — a missing required field will cause the build to fail with a clear error message.
  </Step>

  <Step title="Preview the production build">
    <CodeGroup>
      ```bash npm
      npm run preview
      ```
      ```bash pnpm
      pnpm preview
      ```
      ```bash yarn
      yarn preview
      ```
    </CodeGroup>

    This serves the `dist/` folder locally so you can verify the output before deploying.
  </Step>
</Steps>

## Available scripts

All scripts are defined in `package.json`:

| Script | Command | Description |
|---|---|---|
| `dev` | `astro dev` | Start local dev server at `localhost:4321` |
| `build` | `astro build` | Build static site to `dist/` |
| `preview` | `astro preview` | Serve `dist/` locally |
| `astro` | `astro` | Run Astro CLI directly |
