## Overview

This is the documentation for **Rsantana Portfolio**, a personal portfolio website built by Ronald Santana. The site showcases web development projects using a clean, minimal design powered by [Astro](https://astro.build) and [Bootstrap 5](https://getbootstrap.com).

## Tech stack

The portfolio is built with modern, lightweight tools designed for fast static sites:

| Technology | Purpose |
|---|---|
| [Astro 5](https://astro.build) | Static site framework |
| [Bootstrap 5.3](https://getbootstrap.com) | Responsive CSS framework |
| [Bootstrap Icons](https://icons.getbootstrap.com) | Icon library |
| [Onest Font](https://fonts.google.com/specimen/Onest) | Typography |
| TypeScript | Type-safe configuration |

## Key features

<CardGroup cols={2}>
  <Card title="Content collections" icon="database">
    Projects are managed as Markdown files with type-safe frontmatter schema validation via Zod.
  </Card>
  <Card title="Smooth animations" icon="wand-magic-sparkles">
    CSS `fadeInUp` animations play on page load and as content scrolls into view via IntersectionObserver.
  </Card>
  <Card title="Responsive layout" icon="mobile">
    Bootstrap 5 grid and utility classes ensure the site looks great on all screen sizes.
  </Card>
  <Card title="Static output" icon="bolt">
    Astro builds a fully static site — no JavaScript framework overhead, just fast HTML and CSS.
  </Card>
</CardGroup>
